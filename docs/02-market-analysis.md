# 1. 시장 개요

## 국내 공유오피스·유휴공간 시장

대한민국의 업무공간 시장은 코로나19 이후 급격히 재편되었다.

- 재택근무 확산 → 사무실 공실률 상승
- 긱이코노미(Gig Economy) 성장 → 프리랜서·1인 사업자 증가
- 유연근무제 확대 → 고정 사무실 필요성 감소
- MZ세대 창업 증가 → 초기 비용 절감 수요 폭발

이 흐름은 **유휴 공간을 공유하는 플랫폼**에 강력한 시장 기회를 만들고 있다.

# 3. 고객 세그먼트 분석

## 공급자 (Host) 세그먼트

### Segment A — 중소·중견 기업

| 항목 | 내용 |
|---|---|
| 규모 | 직원 10~200명 |
| 상황 | 재택근무·구조조정으로 빈 책상 발생 |
| 니즈 | 고정비(임대료)를 부분 회수하고 싶다 |
| 의사결정 | 총무팀 또는 대표이사 |
| 수익 가능성 | 책상 1석 기준 월 15~30만원 |

### Segment B — 전문직 사무실

| 항목 | 내용 |
|---|---|
| 유형 | 법무법인, 세무사, 회계사, 노무사, 의원 |
| 특징 | 오전·오후 특정 시간대 빈 공간 존재 |
| 니즈 | 유휴 시간을 수익으로 전환 |
| 추가 가치 | 동종업계 네트워크 형성 가능 |

### Segment C — 스타트업

| 항목 | 내용 |
|---|---|
| 상황 | 급성장 후 이전, 또는 팀 축소 |
| 특징 | 계약 기간 남아 있는 공간 처분 원함 |
| 니즈 | 당장 수익화 필요, 빠른 등록 원함 |

## 이용자 (Guest) 세그먼트

### Persona 1 — 독립 프리랜서

| 항목 | 내용 |
|---|---|
| 직군 | 개발자, 디자이너, 마케터, 번역가 |
| 연령 | 25~40세 |
| 고통 | 집에서 일하면 집중 안 됨, 카페는 불안정 |
| 예산 | 월 10~30만원 |
| 중요 요소 | 조용한 환경, 빠른 인터넷, 근거리 |

### Persona 2 — 1인 기업·스타트업 창업자

| 항목 | 내용 |
|---|---|
| 상황 | 사업자등록은 했으나 사무실 없음 |
| 고통 | 공유오피스 장기계약이 부담 |
| 니즈 | 법인 주소지 등록 가능 여부도 관심 |
| 예산 | 월 20~50만원 |

### Persona 3 — 출장·외근 직장인

| 항목 | 내용 |
|---|---|
| 상황 | 본사가 지방, 서울 출장 시 업무공간 필요 |
| 사용 패턴 | 일 단위, 주 단위 단기 이용 |
| 고통 | 카페·호텔 로비는 집중이 어렵다 |
| 예산 | 일 1~3만원 |

### Persona 4 — 원격근무자

| 항목 | 내용 |
|---|---|
| 상황 | 회사는 다니지만 재택 or 자율 근무 |
| 니즈 | 집 근처 조용한 업무 공간 원함 |
| 특징 | 주 1~3회 이용, 유연한 일정 |

# 4. 경쟁사 분석

## 직접 경쟁사

| 서비스 | 모델 | 강점 | 약점 |
|---|---|---|---|
| **위워크 (WeWork)** | 직접 운영 공유오피스 | 브랜드, 인프라 | 고가, 장기계약, 대도시 집중 |
| **패스트파이브** | 직접 운영 공유오피스 | 국내 1위, 품질 | 고가, 지역 제한 |
| **스파크플러스** | 직접 운영 공유오피스 | 디자인, 서비스 | 장기계약, 제한된 위치 |
| **르호봇** | 지점 운영 | 중소도시 진출 | 품질 편차 |

## 간접 경쟁사

| 서비스 | 모델 | 강점 | 약점 |
|---|---|---|---|
| **네이버 지도 + 카페** | 카페 업무공간 | 접근성, 비용 | 소음, 불안정, 전화 불가 |
| **직방 / 다방** | 부동산 플랫폼 | 인지도 | 장기 임대 중심 |
| **에어비앤비 (Airbnb)** | 주거공간 단기 임대 | 글로벌 브랜드 | 업무공간 특화 없음 |
| **코리아스타트업포럼 커뮤니티** | 비공식 자리 공유 | 무료 | 비체계적, 신뢰 낮음 |

## 경쟁 포지셔닝

빈자리의 목표 포지션: **저가 + 단기계약**

- 기존 공유오피스(위워크, 패스트파이브): 고가 + 장기계약
- 카페: 저가 + 비공식 → 업무공간으로 부적합
- **빈자리**: 합리적 가격 + 유연 계약 + 실제 사무실 환경

**빈자리의 차별화 포인트:**

- 공간을 직접 운영하지 않는다 → 비용 없음
- 전국 어디서나 등록 가능 → 지역 무제한
- 단기·유연 계약 지원 → 부담 없음
- 기존 사무실 분위기 → 집중 근무 환경

# 5. 시장 트렌드

## 공급 측 트렌드

| 트렌드 | 내용 | 빈자리에 미치는 영향 |
|---|---|---|
| 재택근무 정착 | 대기업도 주 2~3일 재택 | 빈 책상 증가 → 공급 확대 |
| 오피스 다운사이징 | 임대 면적 축소 추세 | 유휴 공간 발생 → 공급 확대 |
| 구조조정 | 인원 감소 → 자리 남음 | 공급 확대 |

## 수요 측 트렌드

| 트렌드 | 내용 | 빈자리에 미치는 영향 |
|---|---|---|
| 프리랜서 증가 | 플랫폼 경제 성장 | 이용자 수요 증가 |
| MZ세대 창업 붐 | 부담 없는 공간 원함 | 이용자 수요 증가 |
| 디지털 노마드 | 장소에 구애받지 않는 근무 | 유연한 공간 수요 증가 |
| 워라밸 중시 | 집 근처 업무공간 수요 | 다양한 지역 수요 증가 |

# 6. 규제 및 법률 환경

| 항목 | 내용 | 리스크 수준 |
|---|---|---|
| 건물 용도 제한 | 업무용도 건물만 등록 가능 | 중 |
| 임대차 계약법 | 전대차 동의 여부 확인 필요 | 중-고 |
| 사업자 인증 | 무허가 영업 방지 | 낮음 |
| 개인정보보호법 | 이용자 정보 관리 | 중 |
| 전자상거래법 | 결제·환불 정책 준수 | 낮음 |

핵심 리스크: 임대인(건물주)의 전대차 동의 없이 공간을 공유하는 경우 법적 분쟁 가능성이 있다.
플랫폼 정책으로 **사업자 인증 + 전대차 동의 확인**을 필수 절차로 설정해야 한다.

# 7. SWOT 분석

## Strengths (강점)

- 자산 없는 플랫폼 비즈니스 → 확장 비용 최소화
- 기존 공유오피스 대비 낮은 가격 경쟁력
- 공급자·이용자 모두 WIN-WIN 가치
- 전국 네트워크 확장 가능성

## Weaknesses (약점)

- 초기 공급(Host) 확보 어려움 — 닭이 먼저냐 달걀이 먼저냐 문제
- 브랜드 인지도 없음 (초기)
- 공간 품질 표준화 어려움
- B2B 영업 역량 필요

## Opportunities (기회)

- 재택근무 확산으로 유휴 공간 폭발적 증가
- 프리랜서·1인 기업 가파른 성장
- 공유경제 플랫폼에 대한 긍정적 인식 확산
- ESG 관점에서 유휴 자원 활용 트렌드

## Threats (위협)

- 대형 공유오피스 업체의 유사 서비스 진출 가능성
- 직방·다방 등 부동산 플랫폼의 영역 확장
- 경기 불황 시 예산 축소 → 이용자 감소
- 공간 사고 발생 시 신뢰도 타격

# 8. 시장 진입 전략

## Phase 1 — 거점 확보 (0~6개월)

- **지역:** 서울 강남·마포·성수 집중 공략
- **공급:** 스타트업·IT 기업 집중 영업
- **이용자:** 개발자·디자이너 커뮤니티 바이럴 마케팅

## Phase 2 — 수도권 확장 (6~18개월)

- **지역:** 서울 전역, 경기도 주요 도시
- **공급:** 전문직 사무실(세무사, 법무법인) 온보딩
- **이용자:** SNS 광고, 프리랜서 플랫폼 제휴

## Phase 3 — 전국 확장 (18~36개월)

- **지역:** 부산, 대전, 대구, 광주 등 지방 주요 도시
- **공급:** 지방 법인 영업팀 구성
- **이용자:** 출장자·지방 원격근무자 타겟

# 1. 경쟁 구도 개요

빈자리는 **업무공간 중개 플랫폼**이다.

경쟁 구도는 크게 네 가지 레이어로 나뉜다.

| 레이어 | 유형 | 대표 서비스 |
|---|---|---|
| **직접 경쟁** | 공유오피스 운영사 | 패스트파이브, 스파크플러스, 위워크 |
| **간접 경쟁** | 공간 중개 플랫폼 | 에어비앤비, 직방, 스페이스클라우드 |
| **대체재 (민간)** | 비공식 업무공간 | 카페, 도서관, 호텔 로비 |
| **대체재 (공공)** | 지자체 운영 학습공간 | 마포나루 스페이스 등 구립 공간 |

# 2. 직접 경쟁사 상세 분석

## 2-1. 패스트파이브 (FastFive)

### 개요

| 항목 | 내용 |
|---|---|
| 설립 | 2015년 |
| 운영 모델 | 공간 직접 임차 후 재임대 (운영형) |
| 지점 수 | 서울 중심 약 40개 이상 |
| 주요 지역 | 강남, 홍대, 을지로, 판교 |
| 월 이용료 | 1인 기준 약 30~60만원 |

### 강점

- 국내 1위 브랜드 인지도
- 안정적인 인프라 (회의실, 라운지, 복합 서비스)
- 기업 대상 법인 계약 경험 풍부
- 자체 앱·예약 시스템 보유

### 약점

- 공간 직접 운영 → 고정 임대비 부담 → 높은 이용료
- 수도권 집중 → 지방 이용 불가
- 장기 계약 중심 → 단기 이용 불편
- 공간 수 확장에 물리적 한계

### 빈자리와의 차별점

| 항목 | 패스트파이브 | 빈자리 |
|---|---|---|
| 운영 방식 | 직접 운영 | 플랫폼 중개 |
| 가격 | 월 30~60만원 | 월 10~30만원 (예상) |
| 지역 | 수도권 40개 | 전국 무제한 |
| 계약 단위 | 월 단위 이상 | 일·주·월 유연 |
| 공간 분위기 | 공유오피스 느낌 | 실제 사무실 느낌 |

## 2-2. 스파크플러스 (Sparkplus)

### 개요

| 항목 | 내용 |
|---|---|
| 설립 | 2017년 |
| 운영 모델 | 공간 직접 임차 후 재임대 (운영형) |
| 지점 수 | 서울·경기 약 20개 이상 |
| 주요 지역 | 강남, 서초, 여의도 |
| 월 이용료 | 1인 기준 약 35~70만원 |

### 강점

- 프리미엄 인테리어·디자인
- 대기업·법인 고객 다수
- 고급 편의 서비스 (라운지, 케이터링)
- 기업 맞춤형 전용 공간 제공

### 약점

- 가격이 가장 높은 편
- 개인·소규모 프리랜서 접근 어려움
- 지역 극히 제한적
- 단기 이용 거의 불가

### 빈자리와의 차별점

스파크플러스는 **프리미엄 법인 고객 특화**로 빈자리의 주 타겟(프리랜서·1인 기업)과 겹치지 않는다. 오히려 스파크플러스가 너무 비싸서 이탈하는 고객이 빈자리의 핵심 수요층이다.

## 2-3. 위워크 (WeWork)

### 개요

| 항목 | 내용 |
|---|---|
| 설립 | 2010년 (한국 진출 2016년) |
| 운영 모델 | 공간 직접 임차 후 재임대 (운영형) |
| 글로벌 | 39개국 700개 이상 지점 |
| 국내 지점 | 서울 주요 비즈니스 지구 |
| 월 이용료 | 1인 기준 약 50만원 이상 |

### 강점

- 글로벌 브랜드 인지도
- 글로벌 멤버십 (해외 출장 시 사용 가능)
- 네트워킹 이벤트, 커뮤니티 서비스
- 프리미엄 인프라

### 약점

- 2023년 파산보호 신청 → 신뢰도 타격
- 가장 높은 가격대
- 국내 지점 축소 추세
- 기업문화가 외국계 → 국내 중소기업 접근성 낮음

### 빈자리와의 차별점

위워크의 핵심 고객은 **글로벌 기업·스타트업**이다. 빈자리의 주 타겟인 국내 프리랜서·1인 기업과는 층이 다르다. 위워크의 재정 불안정성은 오히려 **빈자리에 기회**가 된다.

## 2-4. 르호봇 비즈니스 인큐베이터

### 개요

| 항목 | 내용 |
|---|---|
| 설립 | 2012년 |
| 운영 모델 | 프랜차이즈 방식 공유오피스 |
| 지점 수 | 전국 약 200개 이상 |
| 특징 | 지방 도시까지 진출한 유일한 대형 브랜드 |
| 월 이용료 | 약 20~40만원 |

### 강점

- 전국 네트워크 (지방 도시 포함)
- 프랜차이즈 방식으로 빠른 확장
- 상대적으로 합리적인 가격

### 약점

- 지점별 품질 편차 심함
- 인테리어·서비스 수준 낮음
- 브랜드 이미지 노후화
- 공유오피스 특유의 낯선 환경

### 빈자리와의 차별점

르호봇은 **전국망**에서 유일하게 빈자리와 직접 경쟁하는 서비스다. 그러나 운영형이라 품질 표준화가 어렵고 비용도 높다. 빈자리는 같은 지역에서 **더 저렴한 실제 사무실 환경**을 제공할 수 있다.

# 3. 간접 경쟁사 상세 분석

## 3-1. 스페이스클라우드 (SpaceCloud)

### 개요

| 항목 | 내용 |
|---|---|
| 운영 모델 | 공간 단기 임대 중개 플랫폼 |
| 주요 공간 | 파티룸, 스튜디오, 회의실, 연습실 |
| 수익 모델 | 중개 수수료 |
| 이용 단위 | 시간 단위 |

### 강점

- 공간 중개 플랫폼 모델이 빈자리와 유사
- 등록 공간 수 많음
- 시간 단위 유연한 이용

### 약점

- 업무(책상) 공간 특화가 아님 → 파티룸·스튜디오 위주
- 장기 이용 시스템 미비
- 사업자 대상 서비스 부재

### 빈자리와의 관계

스페이스클라우드는 **여가·이벤트 공간** 중심이고, 빈자리는 **일상적 업무공간** 중심이다. 직접 경쟁보다는 **시장을 분할**하는 관계에 가깝다.

## 3-2. 직방 / 다방

### 개요

| 항목 | 내용 |
|---|---|
| 운영 모델 | 부동산 중개 플랫폼 |
| 주요 상품 | 아파트, 오피스텔, 상가 임대 |
| 계약 단위 | 월 단위 이상, 전세·월세 |

### 강점

- 압도적 브랜드 인지도
- 대규모 사용자 기반
- 공인중개사 네트워크

### 약점

- 장기 임대 중심 → 단기·유연 공간 없음
- 사무용 책상 단위 거래 불가
- 공간 공유 플랫폼으로 전환 시 대규모 피벗 필요

### 빈자리와의 관계

직방·다방이 **단기 업무공간 시장에 진출할 위협**은 있다. 그러나 핵심 비즈니스가 달라 단기적으로는 위협이 낮다. 빈자리가 먼저 브랜드를 확보해야 한다.

## 3-3. 에어비앤비 (Airbnb)

### 개요

| 항목 | 내용 |
|---|---|
| 운영 모델 | 숙박 공간 중개 플랫폼 |
| 특징 | P2P 방식, 글로벌 네트워크 |

### 강점

- 글로벌 플랫폼 신뢰도
- 다양한 공간 등록 가능
- UX·결제 시스템 완성도 높음

### 약점

- 업무공간 특화가 전혀 없음
- 사무실 책상 단위 등록 불가
- 비즈니스 인증·보안 기능 없음

### 빈자리와의 관계

에어비앤비는 빈자리의 **서비스 UX 벤치마크**로 삼을 수 있다. 직접 경쟁이 아니라 **레퍼런스 모델**에 가깝다.

# 4. 대체재 분석

## 민간 스터디카페 체인 ⚡ 주목

> 하루 5,900원이라는 파격적인 가격으로 빈자리의 핵심 수요층과 직접 경쟁한다.

### 4-1. 카공족 ([cagongzok.com](https://cagongzok.com/))

| 항목 | 내용 |
|---|---|
| 슬로건 | 국내 최초 라운지형 워크스페이스 |
| 운영 모델 | 프랜차이즈 (가맹점 방식) |
| 지점 수 | **150호점 돌파** (2026년 6월 가맹계약 기준) |
| 성장 속도 | 2025년 3월 6호점 → 2026년 6월 150호점 (1년 3개월 만에 25배 성장) |
| 이용료 | **24시간 5,900원** (하루 단위) |
| 운영 시간 | 24시간 |
| 주요 혜택 | 고급 블랜딩 커피 무제한, 외부음식 반입 가능, 전좌석 콘센트, 2시간 자리비움 서비스 |
| 컨셉 | 카페의 자유로움 + 스터디카페의 집중력 = 라운지형 워크스페이스 |

**강점**

- 하루 5,900원 → 월 30일 이용 시 177,000원 수준 (공유오피스 대비 압도적 저렴)
- 24시간 운영 → 야간·새벽 이용자 흡수
- 커피 무제한 포함 → 카페 대비 가성비 우위
- 폭발적 확장세 → 전국 네트워크 빠르게 구축 중
- 프랜차이즈 모델 → 자본 없이 빠른 확장 가능

**약점**

- 라운지 구조 → 전화 업무·화상회의 어려움
- 스터디카페 분위기 → 비즈니스 미팅 부적합
- 사업자 인증·법인 주소 서비스 없음
- 이용자 수준이 학생~일반인 혼재 → 업무 집중 환경 보장 어려움

### 4-2. 공짜당 ([shop.gongzzadang.com](https://shop.gongzzadang.com/))

| 항목 | 내용 |
|---|---|
| 서비스명 | 공짜당 스터디카페 |
| 이용료 | **하루 5,900원** (커피 무제한 포함) |
| 운영 시간 | 24시간 |
| 결제 방식 | QR 코드 스캔 → 즉시 결제 입실 (키오스크 없음) |
| 지점 호환 | 구매 패스권으로 다른 공짜당 지점 교차 사용 가능 |
| 자리비움 | 최대 2시간 |
| 로열티 | 10회 방문 시 1회 무료 (출석 이벤트) |

**강점**

- QR 결제 방식 → 앱 없이 즉시 이용 → 마찰 없는 UX
- 지점 간 패스권 교차 사용 → 멤버십 충성도 높음
- 24시간 운영 + 5,900원 → 카공족과 동일 가격 포지션

**약점**

- 카공족 대비 지점 수 적음 (전국 커버리지 한계)
- 스터디카페 포맷 → 업무 특화 아님
- 사업자 대상 서비스 없음

---

### 카공족·공짜당이 빈자리에 미치는 영향

| 구분 | 카공족·공짜당 | 빈자리 |
|---|---|---|
| 가격 | 하루 5,900원 | 하루 ~10,000원 (예상) |
| 환경 | 라운지·스터디카페 | 실제 기업 사무실 |
| 전화 업무 | 어려움 | 가능 |
| 비즈니스 분위기 | 낮음 | 높음 |
| 사업자 서비스 | 없음 | 있음 (인증·주소 등) |
| 네트워킹 | 없음 | 있음 (동종업계 교류) |
| 예약 단위 | 당일 즉시 | 일·주·월 유연 |

**핵심 분석:**

카공족·공짜당은 **가격 측면에서 빈자리보다 저렴**하다. 따라서 빈자리는 단순 업무공간 가격 경쟁에서 벗어나야 한다.

빈자리의 차별 포인트는 **"진짜 사무실 경험"** 이어야 한다:

- 전화·화상회의 가능한 환경
- 실제 기업 사무실 분위기 → 비즈니스 신뢰감
- 같은 공간 호스트와 자연스러운 네트워킹
- 사업자 등록 주소로 활용 가능 (장기 이용 시)

→ **5,900원 스터디카페를 쓰다가 "역시 진짜 사무실이 필요해"** 를 느끼는 사람들이 빈자리의 핵심 전환 타겟이다.

## 카페

| 항목 | 내용 |
|---|---|
| 비용 | 아메리카노 한 잔 = 약 4,000~6,000원 |
| 장점 | 접근성, 분위기, 저렴함 |
| 단점 | 소음, 전화 불가, 장시간 눈치, 자리 불안정, 보안 없음 |
| 이용 패턴 | 2~4시간 단기 이용 |

카페는 빈자리의 **가장 큰 대체재**다. 그러나 카페는 장시간 집중 업무에 한계가 있어, 프리랜서·1인 기업의 **고통 포인트**가 된다. 빈자리는 카페 이용자를 전환시키는 것이 초기 수요 창출의 핵심이다.

### 강점

- 압도적으로 저렴한 이용료 (사실상 무료에 가까움)
- 세금으로 운영 → 수익 목표 없음 → 가격 경쟁 불가
- 쾌적한 시설·냉난방 완비
- 지역 주민 친화 → 집 근처 접근성 높음

### 약점

- **주 타겟이 학생** → 업무 목적 성인과 다름
- 25세 이상은 하루 5,000원 → 성인 직장인에게는 덜 매력적
- 사업자 등록, 법인 주소, 전화 업무 불가
- 운영 시간 제한 (공공기관 특성)
- 전국 확산 속도 느림 (예산·행정 한계)

### 빈자리와의 관계

공공 학습공간의 주 타겟은 **10~20대 학생**이고, 빈자리의 주 타겟은 **25~45세 직장인·프리랜서·창업자**다.

표면적으로 수요층이 다르지만, 다음 시나리오를 주의해야 한다:

- 지자체가 업무 공간 버전도 출시할 경우 (ex. "청년 창업공간")
- 25세 이상 이용료 인하 시 가성비 경쟁 가열
- 정치적 이슈로 공공 공간 확대 가속화 가능성

**그러나** 공공 공간이 절대 줄 수 없는 것이 있다:

- 실제 기업 사무실 환경
- 네트워킹·비즈니스 분위기
- 사업자 인증·법인 주소 서비스
- 24시간·주말 운영
- 원하는 지역 선택 자유도

→ 빈자리는 **공공이 못 채우는 틈새(성인 비즈니스 특화)**를 공략해야 한다.

# 7. 전략적 시사점

## 지금 진입해야 하는 이유

- 위워크 파산 → 공유오피스 신뢰도 흔들림
- 재택근무 정착 → 유휴 공간 증가 가속
- 프리랜서·1인 기업 폭발적 증가
- 아직 **플랫폼 중개 모델**로 이 시장에 특화된 서비스가 없다

## 우선 방어해야 할 포지션

> **합리적 가격 × 실제 사무실 환경 × 전국 어디서나**

이 3가지 포지션을 선점하면, 기존 공유오피스 운영사는 비즈니스 모델 특성상 빈자리를 따라오기 어렵다.

## 가장 위험한 시나리오

직방 또는 네이버 부동산이 단기 업무공간 카테고리를 열면 경쟁이 격화될 수 있다.
→ 빠른 공급(Host) 확보와 브랜드 인지도 구축이 핵심 방어 전략이다.

## 핵심 결정 사항

### 제품 방향

- **양면 플랫폼**: Host(공급자) + Guest(이용자)
- **MVP 지역**: 서울 강남·마포·성수 집중
- **MVP 기간**: 3개월 목표

### 수익 모델

- **초기**: 무료 중개 (수수료 없음) — 당근마켓 모델
- **Phase 1**: 공간 상단 노출 광고
- **Phase 2**: Host Premium 구독 (월 19,000 / 49,000원)
- **미래**: All-Access 패스, 버추얼 오피스, 기업 복지 연계

### 핵심 경쟁 우위

- 공간 직접 운영 안 함 → Asset-light
- 전국 어디서나 등록 가능
- 일·주·월 유연한 계약
- 진짜 사무실 분위기 (카공족과 차별화)

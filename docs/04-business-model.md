# 1. Customer Segments (고객 세그먼트)

## 빈자리는 양면 플랫폼(Two-sided Platform)이다

| 구분 | 세그먼트 | 설명 |
|---|---|---|
| **Host (공급자)** | 중소·중견 기업 | 재택근무·구조조정으로 빈 책상이 생긴 기업 |
| **Host (공급자)** | 전문직 사무실 | 세무사, 법무법인, 회계법인 등 유휴 시간대 공간 보유 |
| **Host (공급자)** | 스타트업 | 이전·축소 후 계약 기간이 남은 공간 보유 |
| **Guest (이용자)** | 프리랜서 | 개발자, 디자이너, 마케터 등 독립 근무자 |
| **Guest (이용자)** | 1인 기업·창업자 | 사업자는 있으나 사무실이 없는 초기 창업자 |
| **Guest (이용자)** | 출장·외근 직장인 | 지방 출장 시 단기 업무공간이 필요한 직장인 |
| **Guest (이용자)** | 원격근무자 | 재택근무 중 집 근처 업무공간을 원하는 직장인 |

---

# 2. Value Propositions (가치 제안)

## Host에게

| 가치 | 설명 |
|---|---|
| **수익 창출** | 아무 수익이 없던 빈 책상에서 월 10~30만원 수익 발생 |
| **운영 부담 없음** | 플랫폼이 계약·결제·분쟁을 처리 → Host는 공간만 제공 |
| **전국 이용자 연결** | 플랫폼 마케팅으로 직접 홍보 없이 이용자 유입 |
| **네트워킹** | 다양한 업종의 전문가와 자연스러운 교류 |

## Guest에게

| 가치 | 설명 |
|---|---|
| **합리적 가격** | 공유오피스 대비 30~50% 저렴한 실제 사무실 이용 |
| **유연한 계약** | 일·주·월 단위로 필요한 만큼만 이용 |
| **실제 사무실 환경** | 카페와 달리 전화 가능, 집중 업무 환경, 보안 유지 |
| **즉시 예약** | 지도 기반 검색 → 즉시 예약·결제 |
| **전국 어디서나** | 서울뿐 아니라 지방 도시에서도 이용 가능 |

# 3. Channels (채널)

## 인지 (Awareness)

| 채널 | 전략 |
|---|---|
| 인스타그램·링크드인 | 프리랜서·1인 기업 타겟 콘텐츠 마케팅 |
| 구글·네이버 검색광고 | "공유오피스 저렴한", "단기 사무실" 키워드 |
| 프리랜서 커뮤니티 | 크몽, 프리모아, 긱플러스 제휴·배너 |
| 언론·PR | 스타트업 미디어 (디지털투데이, 플래텀) |

## 획득 (Acquisition)

| 채널 | 전략 |
|---|---|
| 자체 웹/앱 | 지도 기반 검색 UX, 즉시 예약 |
| B2B 직접 영업 | 스타트업·IT 기업 총무팀 직접 접촉 |
| HR 복지 파트너십 | 기업 복지몰과 연계 |

## 유지 (Retention)

| 채널 | 전략 |
|---|---|
| 앱 푸시 알림 | 재방문 유도, 새로운 공간 추천 |
| 이메일 뉴스레터 | 주변 신규 공간 알림 |
| 리뷰·후기 | 신뢰도 구축, 재예약 유도 |

# 4. Customer Relationships (고객 관계)

## Host

| 관계 유형 | 내용 |
|---|---|
| 전담 온보딩 | 초기 공간 등록, 사진 가이드, 가격 설정 지원 |
| 정기 리포트 | 조회수, 예약률, 수익 현황 월간 리포트 |
| 호스트 커뮤니티 | 우수 호스트 표창, 오프라인 네트워킹 행사 |
| 전담 CS | 분쟁·문의 신속 처리 |

## Guest

| 관계 유형 | 내용 |
|---|---|
| 자동화된 경험 | 예약→결제→입장 안내 자동화 |
| 리뷰 시스템 | 이용 후 후기 작성 → 신뢰도 구축 |
| 고객 센터 | 채팅·이메일 CS 운영 |
| 맞춤 추천 | 이용 이력 기반 공간 추천 |

# 5. Revenue Streams (수익 구조)

## 핵심 수익 (Core)

### 중개 수수료

| 항목 | 내용 |
|---|---|
| 수수료율 | 거래액의 **10~15%** |
| 부담 주체 | Host 또는 Guest (협의 설계) |
| 예시 | 월 30만원 공간 → 플랫폼 수익 3~4.5만원 |

## 부가 수익 (Additional)

### 프리미엄 노출 광고

| 상품 | 가격 (예시) | 내용 |
|---|---|---|
| 검색 상단 노출 | 월 3~5만원 | 지역 검색 상위 배치 |
| 추천 공간 배너 | 월 5~10만원 | 메인 화면 추천 영역 |

### Host Premium 구독

| 플랜 | 가격 (예시) | 혜택 |
|---|---|---|
| Basic | 무료 | 기본 등록·예약 관리 |
| Pro | 월 2만원 | 고급 통계, 다중 공간 관리, 우선 노출 |
| Business | 월 5만원 | API 연동, 전담 매니저, 기업 계약 지원 |

### 기업 서비스 (B2B)

| 상품 | 내용 |
|---|---|
| 임직원 복지 연계 | 기업 복지 예산으로 빈자리 이용권 구매 |
| 다수 공간 관리 API | 대형 호스트 전용 통합 관리 솔루션 |

# 9. Cost Structure (비용 구조)

## 고정비

| 항목 | 설명 |
|---|---|
| 플랫폼 개발·유지 | 서버, 앱/웹 개발·운영비 |
| 팀 인건비 | 개발, 영업, 운영, CS 인력 |
| 법무·회계 | 계약 검토, 세무 처리 |

## 변동비

| 항목 | 설명 |
|---|---|
| 마케팅·광고 | 검색 광고, SNS 광고, 제휴 마케팅 |
| CAC (고객 획득 비용) | Host·Guest 각각의 온보딩 비용 |
| PG 수수료 | 결제 처리 수수료 (약 2~3%) |
| CS 운영 | 고객 문의·분쟁 처리 비용 |

## 비용 최적화 원칙

- **Asset-light**: 공간을 직접 임차하지 않는다 → 가장 큰 비용 제거
- **자동화**: 예약·결제·계약을 자동화하여 운영 인력 최소화
- **커뮤니티**: 리뷰·바이럴로 유기적 성장 → CAC 절감

# 10. 유닛 이코노믹스 (Unit Economics)

## 핵심 지표 목표

| 지표 | 설명 | 목표 |
|---|---|---|
| **CAC (Host)** | Host 1명 획득 비용 | < 5만원 |
| **CAC (Guest)** | Guest 1명 획득 비용 | < 3만원 |
| **LTV (Host)** | Host 1명 생애 가치 | > 50만원 |
| **LTV (Guest)** | Guest 1명 생애 가치 | > 20만원 |
| **Gross Margin** | 수익 마진 | > 70% |
| **Take Rate** | 거래액 대비 수수료율 | 10~15% |

## 손익분기점 시나리오

| 항목 | 수치 |
|---|---|
| 월 운영비 (고정) | 약 3,000만원 (팀 10명 기준) |
| 평균 거래액 | 월 25만원/공간 |
| 수수료율 | 10% → 공간당 2.5만원 수익 |
| BEP (손익분기) | 월 1,200개 공간 활성 예약 |

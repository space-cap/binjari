# 빈자리 (Binjari)

> **남는 자리를 연결하다.**
>
> 기업·사무실의 유휴 업무공간을 프리랜서·1인 기업과 연결하는 공간 중개 플랫폼

---

## 프로젝트 소개

빈자리는 재택근무·구조조정으로 비어 있는 사무실 책상을 필요한 사람에게 연결한다.

- **Host(공급자)**: 남는 책상으로 추가 수익
- **Guest(이용자)**: 공유오피스보다 저렴한 진짜 사무실 환경
